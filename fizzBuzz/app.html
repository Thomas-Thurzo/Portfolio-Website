FIZZ-BUZZ Ein Coding Projekt

Eine JavaScript Coding Übung. Schreibe ein Programm das Nummern von 1 bis 100 anzeigt. Für eine gewünschte Zahl und das jeweilige vielfache dieser Zahl soll "Fizz" angezeigt werden und für eine andere gewünschte Zahl und das jeweilige vielfache soll "Buzz" angezeigt werden. Für die Zahlen die das vielfache der ersten und der zweiten Zahl sind soll "Fizz-Buzz" angezeigt werden.
